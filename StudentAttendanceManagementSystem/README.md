# Student Attendance Management System

## How to Run in IntelliJ IDEA
1. Open IntelliJ IDEA.
2. Select **File > Open**.
3. Select this folder.
4. If prompted, mark the `src` folder as the **Sources Root** (Right-click `src` > Mark Directory as > Sources Root).
5. Open `src/com/attendance/Main.java`.
6. Click the Green Play button.
